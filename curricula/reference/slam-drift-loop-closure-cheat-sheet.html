SLAM drift and loop closure

The correction changes global consistency; it must not masquerade as instantaneous vehicle motion

Why drift accumulates

Each relative motion estimate contains error. Later estimates build on earlier ones, so small errors in observations, calibration, timing, translation, and rotation accumulate.

What loop closure adds

Place recognition and geometric verification connect the current observation to an earlier place. This long-range constraint lets optimization revise the trajectory and map.

Front end

Consumes sensor data, tracks features, associates observations, estimates current motion, and creates constraints or keyframes.

Back end

Optimizes poses and landmarks from constraints. It may perform local mapping, loop correction, map merging, or global bundle adjustment.
