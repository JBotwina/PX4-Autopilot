PX4 gimbal frame math

One-page reference for earth yaw, vehicle-relative yaw, and the Gazebo conversion.

Frames

FrameHorizontal axesYaw zero
Earth NEDNorth, EastNorth
Vehicle FRDForward, RightVehicle nose

PX4 yaw is positive clockwise when viewed from above: north 0°, east +90°, south ±180°, west −90°.

Yaw-only conversion

joint = wrap(target − vehicle)
actual = wrap(vehicle + joint)

For vehicle east (+90°) and target north (0°), command the joint to −90°.

Bug signature

broken joint = target
broken actual = wrap(vehicle + target)

The earth-frame target is mistakenly sent to a vehicle-relative Gazebo joint, so vehicle heading leaks into feedback.

Quaternion form

q_joint = inverse(q_vehicle) ⊗ q_target
q_vehicle ⊗ q_joint = q_target

Quaternion multiplication replaces angle addition in 3D. Order matters.

Listener diagnosis

  1. Read vehicle earth yaw from vehicle_attitude.
  2. Read the commanded quaternion and flags from gimbal_device_set_attitude.
  3. Use the flags before interpreting the commanded yaw.
  4. Compare the earth-frame result with gimbal_device_attitude_status.
